# Say QUIZ Bot

Telegram-бот для навчання та проходження тестів (JSON + медіа).

## Вимоги

- Python 3.10+
- `pip` та віртуальне середовище (опційно)

## Установка

```bash
# (опційно) створити та активувати віртуальне середовище
python -m venv .venv
# Windows:
.venv\Scripts\activate
# Linux/macOS:
# source .venv/bin/activate

# встановити залежності
pip install -r requirements.txt
